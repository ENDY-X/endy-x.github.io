                  一阶线性微分方程 #

                  \[ \dfrac{dy}{dx}+P(x)y=Q(x) \] 通解: \[ y=e^{-\int P(x)dx} \left[ \int Q(x) e^{\int P(x)dx}dx + C \right]\] 注意还有的形式是: \[\dfrac{dx}{dy} + P(y)x=Q(y)\]

                  二阶线性常系数齐次方程 #

                  \[ y''+py'+qy=0 \]
                  特征方程 \(r^2+pr+q=0\) 的两个根微分方程 \(y''+py'+qy=0\) 的通解
                  两个不相等的实根\(y=C_{1}e^{r_{1}x}+C_{2}e^{r_{2}x}\)
                  两个相等的实根\(y=(C_1+C_2x)e^{r_1x}\)
                  一对共轭复根 \(r_{1,2}=\alpha \pm \beta i\)\(y=e^{\alpha x}(C_1 \cos \beta x + C_2 \sin \beta x)\)
